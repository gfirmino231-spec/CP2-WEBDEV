### Descrição
Implementação em **JavaScript puro** de quatro cifras históricas usadas na criptografia:

1. **Cifra Atbash** – Substitui A↔Z, B↔Y, etc.  
2. **Cifra de César** – Desloca as letras por um número fixo (a chave).  
3. **Cifra de Vigenère** – Usa uma palavra-chave para criar deslocamentos variados.  
4. **RSA Didático** – Simula criptografia com chave pública e privada usando números pequenos.
