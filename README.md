### Descrição
Este projeto implementa **4 cifras importantes da história da criptografia** usando **JavaScript puro (sem bibliotecas)**:

1. **Cifra Atbash** – Substitui A↔Z, B↔Y, etc.  
2. **Cifra de César** – Desloca as letras com uma chave fixa.  
3. **Cifra de Vigenère** – Usa uma palavra-chave para definir vários deslocamentos.  
4. **RSA Didático** – Simula criptografia com chave pública e privada usando números pequenos.

### Como testar
